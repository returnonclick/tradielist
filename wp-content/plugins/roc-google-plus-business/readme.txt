=== ROC Space Interest ===
Contributors: littleJoey
Requires at least: 4.4
Tested up to: 4.4.2
Stable tag: 1.0

Lets users to register interest in spaces that are not available. When a new space match with an interest the system will notify the user by email.


== Installation ==

To install this plugin, please refer to the guide here: [http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation)

After install the plugin, please create a PAGE called unsubscribe-interest and type the follow shortcode as a content:
[interest_unsubscribe]

Use the shortcode [interest_form] and [interest_button] to put a bar on the screen that will call the interest register form
